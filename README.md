# Lossless ASCII Text Compression Application
A Java program that implements (4) four lossless text compression techniques for ASCII Text and compares compression ratio.

 - Lempel Ziv 1977

 - Lempel Ziv Welch

 - Huffman Encoding

 - Arithmetic Coding
